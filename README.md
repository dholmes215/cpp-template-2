# C++ Template
A demonstration of how to configure a modern C++ project properly.

## Requirements

* **A C++ compiler**

## Building & Running

```
PS C:\git\cpp-template-2> cd src
PS C:\git\cpp-template-2\src> g++ main.cpp -o main
PS C:\git\cpp-template-2\src> ./main
Hello World!
```
